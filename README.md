# Parking Management System

## Description

This project is a Python-based Parking Management System that uses computer vision to detect vehicles, extract license plates, and log entry information automatically. 
The system is designed to assist with automated gate control by identifying cars in real time and recording their arrival details.

## Features

- Vehicle detection using YOLOv8.
- Automatic license plate extraction using OCR.
- Real-time logging of car entries into a CSV file.
- Gate control simulation based on detection status.
